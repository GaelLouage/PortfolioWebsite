📖 StoryHub

Your Daily Dose of Short Stories. A web app offering bite-sized narratives across various genres, powered by SQL and Entity Framework.

🔧 Technologies Used

  • 🌐 ASP.NET MVC: (Web Application Framework)
  • 📊 Entity Framework: (Object-Relational Mapper for data access)
  • 💾 SQL Database: (Relational Database Management System for data storage)
  • 🧩 Dependency Injection: (Built-in architectural pattern for modularity and testability)

📌 Features

  • 🌐 ASP.NET MVC Web Application
  • 🍃 MongoDB integration using generic repositories
  • ✅ Validation layer for consistency checks
  • 🏗️ Scalable and clean service/repository architecture
  • 🚀 Swagger-enabled for API testing
  • 🧩 Dependency injection with scoped services
  • 🔐 JWT-based authentication for secure access control
