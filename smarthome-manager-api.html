Smart Home Manager API

🏠 Smart Home Manager API

A secure, modular ASP.NET Core Web API designed for managing smart home devices and rooms. This backend-only project includes JWT authentication, role-based access, clean architecture, and entity-level caching.

🔧 Technologies Used

  • .NET 7/8: (ASP.NET Core Web API)
  • SQL Server: (Relational database managed via EF Core)
  • Entity Framework Core: (ORM for data access)
  • JWT Authentication: (Token-based authentication system)
  • Swagger / Swashbuckle: (For interactive API documentation)
  • IMemoryCache: (Used for temporary data like login attempts)
  • Dependency Injection: (Built-in via IServiceCollection)

📌 Features

  • 🏠 Manage rooms and smart devices with role-based authorization
  • 🔐 JWT-based authentication with lockout protection for brute-force attacks
  • 📡 RESTful endpoints for CRUD operations on rooms and devices
  • 📄 Swagger-enabled UI for testing and exploring the API
  • 🧱 Clean architecture using Controllers, Services, Repositories, and DTOs
  • 🛠 Extensible service/repository pattern for scalability
  • ⚡ In-memory caching to handle login attempt tracking and account lockouts
